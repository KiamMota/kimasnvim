minha config de edição para o neovim

Linux/MacOs:

``` bash
curl -fsSL https://raw.githubusercontent.com/KiamMota/kimasnvim/main/exec.sh | sh
```

Windows:

``` powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
Invoke-Expression (Invoke-WebRequest -UseBasicParsing "https://raw.githubusercontent.com/KiamMota/kimasnvim/main/exec.ps1").Content
```
